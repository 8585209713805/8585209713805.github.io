# 我的法规学习笔记 · 网站

把用 `statute-study-notebook` 技能生成的法规学习笔记集中管理、按法律领域分类浏览的本地网站。

## 目录结构

```
法规笔记网站/
├── index.html        # 首页：按领域分类的卡片目录 + 搜索 + 筛选
├── add_to_site.js    # 自动登记脚本（技能生成笔记后调用，自动复制+写索引）
├── notes/            # 每篇笔记的 HTML 放这里
└── README.md
```

## 如何使用

1. 在 Finder 打开本目录，双击 `index.html`（用 Chrome / Safari 打开）。
2. 顶部搜索框可按法规名 / 关键词过滤；分类筛选条按领域筛选；左侧栏快速跳转。
3. 点击任意卡片即可打开对应笔记，照常批注、自动保存。

## 新增一篇笔记（两种方式）

- **方式一（推荐，零手动）**：用 `statute-study-notebook` 技能生成笔记时，技能会自动调用本目录的
  `add_to_site.js`，把笔记复制到 `notes/` 并写入 `index.html`，无需你做任何操作。
- **方式二（手动）**：把生成的 `《法规名》学习笔记.html` 复制到 `notes/`，再在 `index.html` 的
  `SITE_NOTES` 数组加一行（含 `cat` 领域字段）即可。

## 自定义站点位置

本站点的位置由 `add_to_site.js` 自身的目录（`__dirname`）决定。若想换位置，
直接把整个 `法规笔记网站/` 文件夹拖动到任意目录即可，脚本会自动以新位置为准；
只需在技能 `SKILL.md` 里把调用路径改成新的 `add_to_site.js` 路径。

## 注意事项

- 笔记批注存在浏览器本地（localStorage），换设备 / 清缓存前，记得在各笔记里点「导出」备份。
- 本站为纯静态文件，可整体复制到其他电脑、或部署到任意静态托管（CloudStudio / GitHub Pages 等）实现公网访问。
