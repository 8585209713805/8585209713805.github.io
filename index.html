关于本网站与配套技能

本网站是一个法规学习笔记的聚合空间：把 AI 生成的「单文件可编辑 HTML 法规笔记」按法律领域分类集中浏览，支持搜索、筛选，手机和电脑都能直接打开。

配套技能 statute-study-notebook：只需说一句「做《XX法》学习笔记」，即可一键生成单文件笔记，包含逐条官方原文、AI 条文解读、分层批注（疑问 / 我的案例 / 要点总结 / 待办）、速查卡、关联真实案例、关联其他法规法条、自测模式，并支持自动保存与导出。生成后会自动登记进本网站。

如何新增一篇笔记？

方式一（推荐）：用 statute-study-notebook 技能生成笔记时，会自动登记进本网站——无需手动复制。

方式二（手动）：把生成的 《法规名》学习笔记.html 复制到本目录的 notes/ 文件夹，然后在下面数据的 SITE_NOTES 数组里加一行（含 cat 领域）：

{title:"法规名", sub:"副标题", file:"notes/文件名.html", tags:["标签"], cat:"所属领域"}

领域建议取值：宪法相关法 / 民商法 / 行政法 / 经济法 / 劳动与社会保障法 / 刑法 / 诉讼与非诉讼程序法 / 司法制度 / 其他。
